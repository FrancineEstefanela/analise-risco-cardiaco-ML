4. Preparao dos dados

Nesta etapa, vamos preparar os dados para os modelos de machine learning, convertendo as variveis e normalizando os valores para anlise mais eficiente. Aps vamos treinar os modelos de Random Forest e Regresso Logstica com os dados de treino, vamos comparar os resultados utilizando o classification report que apresenta as mtricas de avaliao, como preciso, recall, f1-score e acurcia.

In[14]:
# Separao das variveis
X = df_uci.drop('num', axis=1)
y = df_uci['num']

# Padronizao dos dados
X = StandardScaler().fit_transform(X)

# Diviso dos dados em treino e teste (80% treino, 20% teste)
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.28, random_state=42)

4.3 Comparao entre os Modelos

Aps treinar os modelos de Random Forest e Regresso Logstica, vamos comparar os resultados utilizando as mtricas de preciso, recall, f1-score e acurcia.

In[17]:
# Definindo as mtricas a serem comparadas
metrics = ['Precision', 'Recall', 'F1-Score']
rf_scores = [
    0.79,  # Precision para classe 0 (Random Forest)
    0.84,  # Recall para classe 0 (Random Forest)
    0.81   # F1-Score para classe 0 (Random Forest)
]
log_reg_scores = [
    0.88,  # Precision para classe 0 (Logistic Regression)
    0.86,  # Recall para classe 0 (Logistic Regression)
    0.87   # F1-Score para classe 0 (Logistic Regression)
]

# Criando o grfico
x = np.arange(len(metrics))  # A posio das barras
width = 0.3  # Largura das barras

# Criando as barras
fig, ax = plt.subplots()
bars_rf = ax.bar(x - width/2, rf_scores, width, label='Random Forest', color='purple')  # Cor roxa para Random Forest
bars_log_reg = ax.bar(x + width/2, log_reg_scores, width, label='Regresso Logstica', color='indigo')  # Cor roxa escuro para Logistic Regression

# Adicionando rtulos e ttulo
ax.set_xlabel('Mtricas')
ax.set_ylabel('Valores')
ax.set_title('Comparao de Desempenho entre Random Forest e Regresso Logstica')
ax.set_xticks(x)
ax.set_xticklabels(metrics)
ax.legend()

# Exibindo os valores nas barras
def add_values(bars):
    for bar in bars:
        height = bar.get_height()
        ax.annotate(f'{height:.2f}', 
                    xy=(bar.get_x() + bar.get_width() / 2, height),
                    xytext=(0, 3),  # Deslocamento para cima
                    textcoords="offset points",
                    ha='center', va='bottom')

# Adicionando os valores s barras
add_values(bars_rf)
add_values(bars_log_reg)

# Exibindo o grfico
plt.show()

Concluso: A Regresso Logstica apresentou melhor desempenho geral, especialmente em termos de acurcia, recall e F1-score, sendo mais equilibrada. Embora o Random Forest tenha sido bom em preciso para a classe 1, a Regresso Logstica mais indicada para esse problema.

  • Acurcia: A Regresso Logstica obteve melhor desempenho (87% vs. 80%).
  • Preciso: O Random Forest teve preciso ligeiramente melhor para a classe 1 (com doena), enquanto a Regresso Logstica foi melhor para a classe 0 (sem doena).
  • Recall: A Regresso Logstica foi mais equilibrada, com recall superior tanto para a classe 0 quanto para a classe 1.
